# UnBooks 

O **UnBooks** tem como objetivo ajudar estudantes da UnB a encontrarem livros relevantes no catálogo da BCE, reduzindo a sobrecarga de informação e oferecendo recomendações personalizadas.

O projeto está em fase inicial de Engenharia de Requisitos, contendo problema, escopo, stakeholders, requisitos, épicos, histórias de usuário, backlog inicial e slides.
