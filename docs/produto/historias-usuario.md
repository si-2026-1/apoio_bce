# Histórias de Usuário

## HU01 — Login com matrícula

**Descrição:**  
Como estudante, eu quero realizar login utilizando minha matrícula da UnB, para que eu possa acessar o sistema com segurança.

**Critérios de aceitação:**

- o sistema deve solicitar matrícula;
- o sistema deve validar o acesso institucional;
- o sistema deve impedir acesso inválido;
- o sistema deve permitir logout.

---

## HU02 — Responder preferências iniciais

**Descrição:**  
Como estudante, eu quero responder um questionário inicial de preferências, para que o sistema consiga gerar recomendações mesmo no primeiro uso.

**Critérios de aceitação:**

- o sistema deve apresentar perguntas sobre interesses;
- o sistema deve salvar as respostas;
- o usuário deve conseguir editar suas preferências;
- o sistema deve usar as respostas no cálculo inicial de recomendação.

---

## HU03 — Registrar leitura

**Descrição:**  
Como estudante, eu quero registrar livros que já li, para que meu histórico ajude na personalização das recomendações.

**Critérios de aceitação:**

- o sistema deve permitir selecionar um livro;
- o sistema deve registrar a leitura;
- o sistema deve impedir registros duplicados;
- o sistema deve associar a leitura ao usuário.

---

## HU04 — Avaliar livro lido

**Descrição:**  
Como estudante, eu quero avaliar livros lidos com estrelas e comentários, para que o sistema entenda melhor minhas preferências.

**Critérios de aceitação:**

- o sistema deve permitir nota de 1 a 5;
- o sistema deve salvar a avaliação;
- o sistema deve impedir avaliações duplicadas;
- o sistema deve atualizar os dados de preferência do usuário.

---

## HU05 — Controlar privacidade

**Descrição:**  
Como estudante, eu quero escolher se uma leitura será pública ou privada, para que eu tenha controle sobre meu histórico.

**Critérios de aceitação:**

- o sistema deve oferecer opção pública ou privada;
- leituras privadas não devem aparecer no feed;
- o usuário deve poder alterar a configuração.

---

## HU06 — Filtrar livros

**Descrição:**  
Como estudante, eu quero filtrar livros, para que eu encontre obras mais adequadas ao meu objetivo.

**Critérios de aceitação:**

- o sistema deve permitir filtro por assunto;
- o sistema deve permitir filtro por idioma;
- o sistema deve permitir filtro por curso;
- o sistema deve permitir filtro por disciplina;
- o sistema deve permitir filtro por nível de profundidade.

---

## HU07 — Adicionar amigos

**Descrição:**  
Como estudante, eu quero adicionar amigos, para que eu possa acompanhar leituras públicas de pessoas próximas.

**Critérios de aceitação:**

- o sistema deve permitir enviar pedido de amizade;
- o sistema deve permitir aceitar pedido;
- o sistema deve impedir amizade duplicada.

---

## HU08 — Visualizar feed

**Descrição:**  
Como estudante, eu quero visualizar leituras públicas dos meus amigos, para que eu descubra novos livros através da comunidade.

**Critérios de aceitação:**

- o sistema deve exibir apenas leituras públicas;
- o sistema deve ocultar leituras privadas;
- o sistema deve mostrar livros recentes dos amigos.
